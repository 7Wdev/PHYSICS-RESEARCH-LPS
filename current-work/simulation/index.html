Kelvin Simulation

Made by Evan Wallace and Modified by Ade Issawe to serve our needs.

Keymap:

  • Touch the water to create ripples
  • Use the mouse to rotate the camera
  • Use SPACEBAR to pause and unpause
  • Drag the ball to move it around
  • Use the L key to redirect the light
  • Use the G key to toggle gravity

we have manipulated this code, to make it suitable for our experiment.
feel free to use the code through my github ;)
